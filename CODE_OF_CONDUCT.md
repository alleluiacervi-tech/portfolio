# Code of Conduct

## Our Commitment

We are committed to maintaining a respectful, inclusive, and constructive environment for everyone who contributes to this project.

## Expected Behavior

- Be respectful and professional in communication.
- Give actionable, technical feedback.
- Assume good intent and seek clarification before escalating.
- Focus discussions on project outcomes.

## Unacceptable Behavior

- Harassment, discrimination, or personal attacks.
- Repeated disruptive behavior in issues or pull requests.
- Sharing private information without consent.

## Reporting

If you experience or witness behavior that violates this code, report it to the maintainers through a private channel.

## Enforcement

Maintainers may edit, hide, lock, or remove content that violates this code and may restrict participation for repeated or severe violations.

## Scope

This code applies to all project spaces, including issues, pull requests, reviews, and discussions.
